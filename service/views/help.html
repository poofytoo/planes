Goal

Your goal is simple - shoot down the other plane and don't get shot. The player who survives is the winner! If neither planes are shot after 120 turns (or both planes are shot at the exact same time), the game is a tie.

Rules

A plane can be in any one of 5 rows (numbered 0 through 4 from top to bottom). Every turn, a plane can take one of several actions. It can move up or down, changing its row. The rows wrap around, so going up in row 0 will put you at row 4. A plane can charge, increasing the myAmmo count. A plane can shoot, which fires a bullet at the other plane, and decreases the myAmmo count. Planes must charge before shooting, and can charge as many bullets as they like.

Keep in mind, because these planes are lightyears apart, shooting a bullet will take exactly 6 turns to reach the other plane. You start in either the top left corner or the bottom right corner (it'll be random). You can see a sample game here!

Tips and Tricks

Because there are no ways for you to save variables or save states, use the gameRound and myLastMove variables wisely!

You can import random and import itertools, but that's just about all the modules you can import. No sketchy/evil things please- we look at your code manually every once in a while to prevent shenanigans ;)

The Code

compute()

This is the main function that you must fill out. Each turn, this function is called to determine what your plane's action should be. You take in the parameters: gameRound, enemyRow, myRow, radars, myAmmo, and myLastMove. You return an up, down, charge, or shoot, otherwise your default action will be to charge.

gameRound int

The current round of the game. Each game lasts exactly 120 rounds (meaning that compute) will be called 120 times. The gameRound is 0-indexed (so the first round is 0).

enemyRow int

The row the enemy is currently occupying, at the beginning of the turn. The first row is 0, and the last row is 4.

myRow int

The row you're in, at the beginning of the turn. The first row is 0, and the last row is 4.

radars list

A Python list containing either SAFE or WARNING for each value. Each index pertains to a row, and will display SAFE if there are no enemy bullets coming at you, or WARNING if there are one or more enemy bullets coming at you.

myAmmo int

The number of bullets you have. Don't spend it all in one place now!

myLastMove string

A string indicating the move you took last turn, which is one of up, down, charge, shoot, or None (in the case of your first turn).
