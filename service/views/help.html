The Game

Overview

Planes is a 1v1 game where your plane, running your code, challenges another player's plane. Each round consists of one action (moving up, down, charging, or shooting). Your code will tell the plane what to do each turn.

Goal

Your goal is simple - shoot down the other plane and don't get shot. The player who survives is the winner! In the case of a tie (either no one is shot in 100 turns, or both planes are shot at the same time), the plane with the most bullets remaining wins!

Rules

A plane can be in any one of 5 rows (numbered 0 through 4 from top to bottom). Every turn, a plane can take one of several actions. It can move up or down, changing its row. The rows wrap around, so going up on the uppermost row (row 0) will cause you to end up in the bottommost row (row 4). A plane can charge, increasing the myAmmo count. A plane can shoot, which consequently fires a bullet at the other plane, and decreases the myAmmo count. If planes shoot with no ammo, then nothing will happen for that turn.

Keep in mind, because these planes are lightyears apart, shooting a bullet will take exactly 6 turns to reach the other plane. You can see a sample game here!

Tips and Tricks

Because there are no ways for you to save variables or save states, use the gameRound and myLastMove variables wisely!

You can import random, but that's just about all the modules you can import. No sketchy/evil things please- we look at your code manually every once in a while to prevent shenanigans ;)

The Code

compute()

This is the main function that you must fill out. Each turn, this function is called to determine what your plane's action should be. You take in the parameters: gameRound, enemyRow, myRow, radars, myAmmo, and myLastMove. You return an up, down, charge, or shoot, otherwise your default action will be to charge.

gameRound int

The current round of the game. Each game lasts exactly 100 rounds (meaning that compute) will be called 100 times.

enemyRow int

The row the enemy is currently occupying, at the beginning of the turn. The first row is 0, and the last row is 4.

myRow int

The row you're in, at the beginning of the turn. The first row is 0, and the last row is 4.

radars list

A Python list containing either safe or warning for each value. Each index pertains to a row, and will display safe if there are no enemy bullets coming at you, or warning if there are one or more enemy bullets coming at you.

myAmmo int

The number of bullets you have. Don't spend it all in one place now!

myLastMove string

A string indicating the move you took last turn, which is one of up, down, charge, shoot, or None (in the case of your first turn).
