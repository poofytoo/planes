Goal

Your goal is simple - shoot down the other plane and don't get shot. The player who survives is the winner! If neither planes are shot after 120 turns, the game is a tie.

Rules

A plane can be in any one of 5 rows (numbered 0 through 4 from top to bottom). Every turn, a plane can take one of six actions: up, down, charge, shoot, supershoot or shield.

two planes on the battlefield with rows numbered

up or down changes the current row of your plane. The rows wrap around, so going up in row 0 will put you at row 4.

charge increasing your myEnergy count. You need energy to shoot, supershoot, or shield, and you can charge as much energy as you'd like throughout the game.

shoot fires a bullet at your opponent (requires 1 energy). Keep in mind, because these planes are lightyears apart, shooting a bullet will take a couple of turns to reach the other plan.

supershoot fires three bullets in parallel at your opponent (requires 5 energy). This is similar to the regular shoot, except you fire two additional bullets at your opponent: one in the row directly above your current row and another below.

the almighty supershoot

shield creates a barrier in front of your plane (requires 3 energy), and extends to the row above and below your current row. Each shield you create will block you from one enemy bullet. Once an bullet hits any part of your shield, your shield is destroyed. You can shoot through your own shield without destroying it.

Tips and Tricks

Because there are no ways for you to save variables or save states, use the gameRound and myLastMove variables wisely!

You can import random and import itertools, but that's just about all the modules you can import. No sketchy/evil things please- we look at your code manually every once in a while to prevent shenanigans ;)

The Code

Overview

To code your AI, you should download the bot template (.py). At the moment, we only support Python (our engine runs Python 2.7.3, so as long as you're not using Python 3.x.x you should be fine!). Once you're done with your code, login, and upload your bot!

compute()

This is the main function that you must fill out. Each turn, this function is called to determine what your plane's action should be. You take in the parameters: gameRound, enemyRow, myRow, radars, myEnergy, and myLastMove. You return an up, down, charge, or shoot, otherwise your default action will be to charge.

gameRound int

The current round of the game. Each game lasts exactly 120 rounds (meaning that compute) will be called 120 times. The gameRound is 0-indexed (so the first round is 0).

enemyRow int

The row the enemy is currently occupying, at the beginning of the turn. The first row is 0, and the last row is 4.

myRow int

The row you're in, at the beginning of the turn. The first row is 0, and the last row is 4.

radars list

A Python list containing either SAFE or WARNING for each value. Each index pertains to a row, and will display SAFE if there are no enemy bullets coming at you, or WARNING if there are one or more enemy bullets coming at you.

myEnergy int

The number of bullets you have. Don't spend it all in one place now!

myShield int

The layers of shield you currently have.

myLastMove string

A string indicating the move you took last turn, which is one of up, down, charge, shoot, or None (in the case of your first turn).
